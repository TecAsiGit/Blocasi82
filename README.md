# Blocasi82

Bloc de notas minimalista, 100% offline y sin límites.  
Lee, modifica y comparte archivos TXT, HTML y CSV con total privacidad.

## Características

- 🔒 **100% offline** – Sin conexión a internet, sin telemetría, sin anuncios.
- 📝 **Sin límites** – Escribe todo lo que quieras, sin restricción de caracteres.
- 📂 **Abre archivos** – TXT, HTML y CSV se muestran como texto plano, sin interpretar.
- 📤 **Comparte como .txt** – Genera un archivo .txt real y compártelo con cualquier app.
- 🌙 **Interfaz oscura** – Cómoda para la vista, adaptable a móvil.
- 💾 **Guardado automático** – Tus notas se guardan solas dentro de la app.
- 📱 **Multiplataforma** – Funciona en Android (APK) y como PWA en PC.
- 🛡️ **Sin permisos innecesarios** – Solo pide acceso a archivos cuando tú decides abrir uno.

## Instalación

Descarga la APK desde la sección [Releases](https://github.com/TecAsiGit/Blocasi82/releases) e instálala en tu Android.

## Uso

1. Escribe sin límites. Todo se guarda automáticamente.
2. Usa 📂 para abrir archivos TXT, HTML o CSV desde tu dispositivo.
3. Usa 📤 para compartir la nota actual como archivo .txt.
4. Tus notas se conservan incluso si cierras la app.

## Privacidad

Blocasi82 no recopila ningún dato. Todo el contenido se almacena exclusivamente en tu dispositivo. No se realizan conexiones a internet.
